# Curso de Algoritmos — Curso em Vídeo  
**Professor Gustavo Guanabara**

Repositório com **exercícios e práticas** desenvolvidos durante o **Curso de Algoritmos** do *Curso em Vídeo*.

O foco deste repositório é **fortalecer a lógica de programação**, base essencial para o aprendizado de qualquer linguagem e para o desenvolvimento de soluções computacionais.

---

## Conteúdos

- Entrada e saída de dados  
- Variáveis e tipos de dados  
- Operadores aritméticos, relacionais e lógicos  
- Estruturas condicionais  
- Estruturas de repetição 
- Procedimentos e funções  
- Vetores  
- Matrizes  
- Lógica aplicada à resolução de problemas

- **Portugol**
